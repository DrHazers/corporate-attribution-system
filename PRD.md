# PRD - Corporate Attribution System

## 1. 项目目标
构建一个面向产业研究的企业国别归属与业务结构征订系统，用于识别企业注册地、上市地与实际控制地之间的差异，并支持对企业业务线进行研究导向的结构化标注。

## 2. 用户角色
### 2.1 研究人员
负责查看企业数据、控制关系和国别归属结果。

### 2.2 标注人员
负责对企业国别归属和业务线进行人工征订与修正。

### 2.3 管理员
负责管理系统数据、查看日志和维护基础规则。

## 3. 核心功能模块

### 3.1 企业信息管理
- 新增企业
- 编辑企业
- 查询企业
- 查看企业详情

### 3.2 股权结构管理
- 录入股东关系
- 维护持股比例
- 支持多层股权结构

### 3.3 控制关系识别
- 识别直接控制关系
- 识别间接控制关系
- 展示控制链条

### 3.4 实际国别归属判定
- 存储注册地、上市地、实际控制地
- 基于控制关系进行判定
- 标记特殊结构（VIE、红筹、跨国控股）
- 输出判定依据

### 3.5 业务线标注
- 记录企业主营业务
- 记录企业新兴业务
- 支持多业务线归属
- 支持产业分类映射

### 3.6 征订与日志
- 记录人工修改
- 保留历史版本
- 保留修改原因
- 支持审计追踪

### 3.7 LLM 辅助模块（后续）
- 企业业务描述抽取
- 业务线分类建议
- 国别归属解释生成

## 4. 非功能需求
- 结果可解释
- 数据可追溯
- 支持版本管理
- 支持未来扩展
- 代码结构清晰，便于维护

## 5. 当前阶段开发范围
第一阶段重点实现后端核心能力，不优先开发复杂前端页面。
